UROP Data Analysis Code are various ways of plotting the data.
UROP Data Download Code stores a program to download Cluster data from the server.
UROP Data Sorting Code takes in downloaded data and turns it into usable .npy arrays for the other code.
UROP Machine Learning Analysis Code contains all the ML models trained for the project.
The Output File Structure Raw file contains the format for the output folders that some of the code uses.

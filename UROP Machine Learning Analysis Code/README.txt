UROP KNN Analysis Code contains all code related to training and testing the KNN models used.
UROP SVR Analysis Code contains all code related to training and testing the SVR models used.
For each of these folders, see the annotation in the title for what the code specifically does. 
Each new version listed is a new save file created as the code evolved: a later version may not have the same functionality as a previous version.
UROP ML OnPeriod Analysis are tests of ML model predictions against the reference values in time. (Figures 22-24)
All the inputs required in the models are found in the inputs folder in each folder.
